Important - Pre Activity
  1. Please verify all the course codes before paying the fees.
  2. The fees will be based on the fee set by the university
    • Income limit
    • Gender
    • Category
    • Reservation claim status of Student
    • Amount will be visible after university sets the fee.
  3. QP Codes need to be set for all courses in the curriculum.
  4. The registered curriculum of the student should be approved by the Principal and also the admission should be approved by the Registrar.
  5. Application fee paying between start date of application pay and last date of application fee (2nd Penalty) set in exam notification.
  6. If the pay button is missing in new application, here might be the following reasons:
    • Curriculum not set.
    • Curriculum not approved.
    • QP Codes not set for all curriculum.
    • Exam fee was not set by the university.
  7. Course registration for Eligible notification is mandatory to apply for the exam.
  8. Special case late fee has to be set in general fee and special case last date penalty in exam notification has to be set.
